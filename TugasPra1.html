Pengertian HTML

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat halaman web. HTML berfungsi untuk menyusun struktur konten di web.

Struktur HTML Dasar

HTML memiliki struktur dasar seperti berikut:

            <!DOCTYPE html>
            <html>
                <head></head>
                <body></body>
            </html>
